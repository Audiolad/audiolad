## Компоненты

| Компонент | Как работает |
|-----------|--------------|
| Next.js | PM2-процесс `audiolad`, порт `3000` |
| Nginx | Проксирует домен на Next.js |
| Supabase | Docker на том же сервере |
| Supabase Studio | http://72.56.232.160:8000 |

## npm-скрипты

| Команда | Действие |
|---------|----------|
| `npm run dev` | Локальная разработка |
| `npm run build` | Production-сборка |
| `npm run start` | Запуск production |
| `npm run lint` | ESLint |

## Перезапуск приложения

```bash
pm2 restart audiolad
```

Выполнять **только после подтверждения** владельца или архитектора.
