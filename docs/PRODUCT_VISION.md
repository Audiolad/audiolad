Продуктовое видение «АудиоЛад».

## Известное позиционирование

Согласно метаданным приложения и UI, «АудиоЛад» — платформа аудиопрактик, медитаций и энергетических программ.

На главной странице указано: «Медитации, энергопрактики и молитвы для любви, изобилия, счастья».

## Каналы привлечения

MAX является одним из приоритетных каналов привлечения пользователей «АудиоЛада».

Платформа рассматривается не только как самостоятельный веб-сервис, но и как продукт, с которым пользователь может впервые познакомиться внутри экосистемы MAX.

## Воронка знакомства через MAX (планируется)

Предполагаемый путь пользователя:

1. Реклама или контакт в MAX.
2. Общение с ботом (в том числе через BotHelp).
3. Получение сообщений с бесплатными аудиопрактиками, которые можно прослушать прямо в чате.
4. Переход по кнопке в мини-приложение «АудиоЛад» внутри MAX.
5. Автоматическое создание или вход в аккаунт на основе подтверждённых данных пользователя MAX.
6. Попадание в личную библиотеку на платформе.

Этот сценарий **не реализован**. Описан как продуктовое направление для последующей разработки.

## Ключевые пользовательские сценарии (из реализованного UI)

- Прослушивание аудиопрактик и программ.
- Регистрация и вход для сохранения коллекции.
- Просмотр каталога практик по категориям.
- Управление плейлистами, избранным, историей прослушивания.
- Покупка платных практик (UI оформлен, оплата не подключена).
- Кабинет автора для публикации практик (UI оформлен, backend не подключён).

## Открытые вопросы по интеграции с MAX

Перед реализацией воронки через MAX необходимо отдельно исследовать:

- официальный MAX Bot API;
