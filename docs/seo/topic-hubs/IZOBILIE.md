# SEO-хаб «Изобилие»

URL: `/topics/izobilie`  
Ветка: `feat/seo-topic-hub-izobilie`

## Модель фильтрации

**`topicKey: "money"` + ordered `practiceSlugAllowlist` (3 slug)**  
`resolveTopicChips: false` – chips `money` остаются на `/catalog?topic=money`.

Analytics:

- `topic_slug` / `hub_slug` = `izobilie`
- `topic_key` = `money`

## Аудит

Явный интент изобилия (title/slug):

1. `klyuch-k-izobiliyu` – free, Квант-Медитация ← точка входа  
2. `aktivatsiya-kanala-izobiliya` – paid, Энергопоток-активация  
3. `koding-izobiliya` – paid, Программа энергопотоков  

Дополнительные кандидаты с «изобилие» только в description (денежный угол) **не включаем** – они уже в `/topics/meditatsii-na-dengi`.

## Достаточность контента

**Да для focused SEO-хаба:** есть free entry + mid paid + deep program, уникальный интент, уже разведены с денежным хабом.  
Риск thin content снижен уникальным copy и FAQ про различие интентов.

## Исключения

Все денежные allowlist-практики из `meditatsii-na-dengi`; пересечение с free-хабом только через `klyuch-k-izobiliyu` (разные интенты страниц).
